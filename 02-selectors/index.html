Bu derste selector kullanımını öğreneceğim.

Bu yazı, stilini etikete verilmiş genel stilden almakta.

Bu yazı, stilini class ile almakta.

Bu yazı, stilini id ile almakta.

Notlarım: Temel olarak bir etikete 3 şekilde stil verilebilir. Birincisi, tüm etikete stil verilebilir. p etiketini örnek verirsek, sayfada p etiketiyle yazılmış olan her şey verdiğimiz stili alır. Farklı p etiketlerine farklı stil vermek istiyorsak; class ya da id kullanıyoruz. Etikete class ismi verdikten sonra stil dosyasında .stiladi şeklinde belirtip stilini değiştirebiliriz. Id vermek istiyorsak yine id adı verdikten sonra stil dosyasında #idadı şeklinde kullanabiliriz.
